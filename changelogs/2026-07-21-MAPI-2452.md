## What Changed

- Replaced the deprecated swagger-repo toolchain with @redocly/cli for linting, bundling, and static ReDoc generation.
- Inject code samples from `spec/code_samples/` into operations as `x-codeSamples` at build time so language tabs still appear in the docs.
- Updated CI to lint then build static output into `web_deploy` and deploy with the existing GitHub Pages artifact flow.
- Documented the repository folder structure and the new npm scripts in the README.

## Why

Move API documentation tooling onto Redocly before swagger-repo / Apiary end of life.

- `scripts`

- `spec` (build-time code sample injection)
